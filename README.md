# stickytab
fully working New Tab page with tools designed for productivity  

Since this New Tab page was designed for computer productivity, it has been designed for a 16:9 screen ratio. It is not recommended to set this as a default New Tab page for any mobile device, or for any device with a screen that is not a 16:9 (landscape) ratio. It is recommended to use a browser extension to set this page as a default newtab page. View this page at https://kpoovakan.github.io/stickytab  

## features
* Stickies: sticky notes, saved locally in your browser, on the left-hand side
* Date and time (24-hour time format) at the top
* Upon loading the page, jumping to a default search bar for web addresses
* Upon loading the page, pressing the TAB key will jump to the search bar for web searches
* Search the web with Google, Bing, or DuckDuckGo (Stickytab is not affiliated with Google, Bing, or DuckDuckGo)
* Planned features: calculator, option to save shortcuts for web addresses, customization
